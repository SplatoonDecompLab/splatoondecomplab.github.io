Silverlight Beta 0.4

This is the fourth beta release of Silverlight and the largest one so far. Note that a lot of new modules are experimental and may not work properly. Below are all the changes made to the mod menu in this version:

Module state saving support is here!

The mod menu now has a "Save Module State" button which saves all of the module states. This means that if you enable a module and then close the game, the module will still be enabled when you decide boot up the game again. Be cautious when using this functionality as you may have to reset your save config if you use it while a module that crashes in the plaza is enabled since that will cause your game to crash on boot.

  • Splatfest Button - Queue splatfest at any time even if one isn't active.

Module Adjustments

  • Always Movable - Camera will always follow the player's camera, including the intro.
  • Always Movable - Movement will always be possible when specials that normally restrain your movement are active.
  • Several miscellaneous adjustments have been made to improve the usage of the mod menu.

Game Fixes

  • Fixed a critical game vulnerability.
  • Fixed invalid object crash.
  • Fixed story mode objects in VSGame crashes.
  • Fixed 1v7 match crash when being the host. (Support for customizable team permutations will be added in a later update.)

Menu Fixes

  • Fixed an issue that caused Silent Splat Bombs to be always enabled.
  • Fixed a critical memory leak related to cache invalidation.
  • Auto-enabled modules now work properly.
  • The "Quit Game" button now properly closes your game instead of crashing it.
