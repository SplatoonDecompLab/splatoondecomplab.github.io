Unused First Fest

There are some leftover BOSS fest files that exist in the Pre-Launch Review editions of Splatoon. These files could be recovered because those editions use separate BOSS servers from the actual game. They are not normally accessible to players as they contain impossible start and end dates. Furthermore, the fest files appear to be an in-development version of the Rice vs. Bread splatfest. Unlike the actual fest, it contains a lot of broken English and unused splatfest tees. Additionally, this is considered the first ever fest that has ever happened because it uses a fest ID of 1. This means that this is the data of the first ever splatfest that was created for Splatoon. Below is all of the datamined information.
