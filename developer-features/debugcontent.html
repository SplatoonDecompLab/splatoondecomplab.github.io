DebugUnderPilot

DebugUnderPilot is a directory that was used by the developers in Splatoon to test the game by automating input. Specifically, all of the code for it is written in Lua, and they appear to use their own Lua framework as a base. We know what it is used for because of Splatoon 2 Global Testfire containing the entire Splatoon 1 Testfire DebugUnderPilot code. Additionally, other Wii U and Switch games hint at how the remaining files that do not exist in Splatoon 2 would look like.

DbgSetting

In Splatoon 2, there exists a complete debug menu that lets you easily access any scene of the game with any target map. This debug menu has several references in Splatoon 1, but no code directly related to it has been found. It is very likely that all of its functionality has been stripped off of the production binary because of conditional compilation.

One notable mention of it is in SeqMgrVersus, where switching to it is used as a case in a switch statement.

DebugMenu

The SoundProcessCheck param in GameConfigSetting clearly suggests that there is another debug menu called "DebugMenu":

ラウドネス等計測モード：0=無効、1=シーン自動計測、2=DebugMenuかORで手動計測（Develope以下限定。変更後は要ロム再起動）

Additionally, in old versions of Splatoon 1 including the testfires, several of the params have the attribute "isOnDebugMenu" set to true, being further proof of this debug menu's existence.

DebugMoving

In a Splatoon gameplay video by GameXPlain, a debug menu with the header "DebugMoving" was accidentally shown. It appears that this debug menu does not exist in the production release of Splatoon. However, we did manage to restore its functionality as we found the debug function responsible for enabling it.

OR

The SoundProcessCheck param in GameConfigSetting says that it can be enabled by using OR. In Splatoon 2, this is a command-based debug feature. This may also exist in Splatoon. Further investigation is required.

Game ROM Types

The game has several ROM types and the game frequently reads the current one to determine how to function.

ROM Type Use
0 - Default
1 - ShowFloor Used in an unreleased demo version of Splatoon and in Splatoon LAN tournaments that were hosted by Nintendo. Makes the game enter LAN mode.
2 - ShowDL Used in Splatoon Testfire. Connects to the official testfire servers.
3 - Kiosk Used in retail stores that have Splatoon playable.
